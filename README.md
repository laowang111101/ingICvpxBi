# 公交车信息管理系统 java459

## 项目概述

本项目是一个基于SSM（Spring + SpringMvc + Mybatis）框架与Vue.js的前后端分离的公交车信息管理系统。采用MVC三层体系结构，结合Maven进行项目管理，实现了Web系统的B/S架构。

## 技术栈

- **前端：** Vue.js
- **后端：** Spring、SpringMvc、Mybatis
- **数据库：** MySQL
- **项目管理：** Maven

## 功能模块

- 公交车信息管理
- 路线规划与查询
- 时间表管理
- 用户权限控制

## 系统角色

- 系统管理员
- 公交车司机
- 普通用户

## 运行环境

- **操作系统：** Windows/Linux/MacOS
- **Java版本：** JDK 1.8及以上
- **数据库：** MySQL 5.7及以上
- **Web服务器：** Tomcat 8.0及以上

## 部署步骤

1. 搭建开发环境，包括JDK、MySQL和Tomcat等。
2. 使用Maven编译项目。
3. 导入数据库脚本。
4. 部署后端代码至Tomcat。
5. 前端代码部署至Web服务器。
